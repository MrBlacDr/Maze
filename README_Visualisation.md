# Лабиринт

С помощью библиотеки *Tkinter* был реализован графический интерфейс (исполняемое приложение создаются по скрипту `gui.py`).
Есть возможность выбирать длину и ширину лабиринта, а также загружать из имеющихся. \
Программа позволяет задавать начальную и конечную точки для нахождения единственного пути (реализовано **рекурсивно**).

# Reinforcement learning

С помощью обучения с подкреплением разработан алгоритм обучения агента кратчайшему прохождению лабиринтов:
- Пользователем указывается файл, в котором описана структура лабиринта, и конечная точка
- Агент способен находить выход из лабиринта из любой начальной точки
- Ичпользовать вкладку ML agent (с выбором предобученных агентов)

В примере показано решение для обученного агента (находит путь из любой точки лабиринта к точке с координатами `{3;4}`)
