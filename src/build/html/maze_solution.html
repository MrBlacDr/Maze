maze_solution module

Решением лабиринта считается кратчайший путь от заданной начальной точки (ячейки таблицы) до конечной. При прохождении лабиринта можно передвигаться к соседним ячейкам, не отделенным «стеной» от текущей ячейки и находящимся сверху, снизу, справа или слева. Кратчайшим маршрут считается, если он проходит через наименьшее число ячеек.

maze_solution.add_path_to_image(image_to_read: str, row: int, col: int, line: list, image_to_save: str) → None[исходный код]

Функция, рисующая путь поверх имеющейся картинки лабиринта.

Параметры:

  • image_to_read – путь к картинке с лабиринтом для чтения

  • row – высота лабиринта

  • col – ширина лабиринта

  • line – список координат точек

  • image_to_save – путь к картинке с нарисованным маршрутом

maze_solution.find_path(maze: list, start: tuple, end: tuple, path: list, not_found=True) → tuple[исходный код]

Рекурсивная функция поиска пути.

Параметры:

  • maze – матрица лабиринта

  • start – координаты точки начала

  • end – координаты конечной точки

  • path – список координат, куда записывается маршрут

  • not_found – флаг, сигнализирующий о завершении поиска

Результат:

кортеж из флага, что найден путь, и список координат самого пути

  • img_read – название картинки для считывания лабиринта

  • img_out – название картинки для сохранения нарисованного маршрута
